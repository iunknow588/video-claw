## 一、需求分析

### 1.1 项目背景

基于技术评审结论，本项目从"全自动内容工厂"调整为**"AI辅助内容创作工具"**。核心目标是通过AI技术提升短视频内容生产效率，同时规避合规风险和技术瓶颈。

### 1.2 核心需求

| 需求编号 | 需求名称 | 优先级 | 说明 |
|----------|----------|--------|------|
| R001 | 热点内容采集 | P0 | 支持抖音、小红书、B站热门内容监控 |
| R002 | AI内容分析 | P0 | 基于DeepSeek-V4分析爆款内容框架 |
| R003 | 脚本生成 | P0 | 基于GLM-5.1生成原创视频脚本 |
| R004 | 视频生成 | P0 | 基于Seedance 2.0生成视频素材 |
| R005 | 人工审核 | P0 | 关键节点人工介入审核 |
| R006 | 半自动发布 | P1 | 人工确认后辅助发布 |
| R007 | 数据监控 | P1 | 基于官方后台数据的效果追踪 |
| R008 | 成本控制 | P0 | 实时监控API调用成本 |

### 1.3 非功能性需求

| 需求编号 | 需求名称 | 指标 |
|----------|----------|------|
| NFR001 | 日产量 | 5-10条精品视频 |
| NFR002 | 视频时长 | 5-15秒 |
| NFR003 | 成本控制 | 单条成本≤50元 |
| NFR004 | 合规性 | 100%人工审核后发布 |
| NFR005 | 可用性 | 系统可用性≥99% |

### 1.4 约束条件

- **合规约束：** 禁止全自动爬取和发布，必须通过官方API或人工操作
- **成本约束：** 月度总成本控制在3万元以内
- **技术约束：** 视频生成依赖Seedance 2.0 API，需处理异步回调
- **人员约束：** 初期团队3-4人，含1名内容审核员

#### 3.2.2 技术实现
```python
def analyze_content(content: HotspotItem) -> AnalysisReport:
    """
    分析爆款内容
    
    Args:
        content: 热点内容
    
    Returns:
        AnalysisReport: 分析报告
    """
    # 1. 构建分析Prompt
    prompt = f"""
    分析以下爆款视频的内容框架：
    标题：{content.title}
    播放量：{content.view_count}
    点赞数：{content.like_count}
    
    请提取：
    1. 内容结构（开头-中间-结尾）
    2. 情绪曲线设计
    3. 钩子设计（前3秒）
    4. 可复用的内容框架
    """
    
    # 2. 调用DeepSeek-V4 API
    response = deepseek_client.chat.completions.create(
        model="deepseek-v4",
        messages=[{"role": "user", "content": prompt}]
    )
    
    # 3. 解析并结构化报告
    return parse_analysis_report(response.choices[0].message.content)
```

### 3.3 脚本生成模块

#### 3.3.1 功能描述

#### 3.3.2 技术实现
```python
def generate_script(analysis: AnalysisReport, 
                   content_type: str,
                   topic: str) -> Script:
    """
    生成视频脚本
    
    Args:
        analysis: 内容分析报告
        content_type: 内容类型
        topic: 主题
    
    Returns:
        Script: 视频脚本
    """
    # 1. 构建生成Prompt
    prompt = f"""
    基于以下爆款内容框架，创作一个原创视频脚本：
    
    框架参考：{analysis.framework}
    内容类型：{content_type}
    主题：{topic}
    
    要求：
    1. 完全原创，不抄袭参考内容
    2. 时长控制在15-30秒
    3. 包含画面描述和旁白文案
    4. 符合平台内容规范
    """
    
    # 2. 调用GLM-5.1 API
    response = glm_client.chat.completions.create(
        model="glm-5.1",
        messages=[{"role": "user", "content": prompt}]
    )
    
    # 3. 解析脚本结构
    return parse_script(response.choices[0].message.content)
```

def generate_video(script: Script, 
                  style: str = "realistic") -> VideoTask:
    """
    生成视频
    
    Args:
        script: 视频脚本
        style: 视频风格
    
    Returns:
        VideoTask: 视频生成任务
    """
    # 1. 构建视频生成Prompt
    video_prompt = f"""
    {script.scene_description}
    风格：{style}
    时长：{script.duration}秒
    """
    
    # 2. 调用Seedance 2.0 API（异步）
    task = seedance_client.video.generations.create(
        model="seedance-2.0",
        prompt=video_prompt,
        size="1080x1920",
        callback_url="https://api.yourapp.com/callback/video"
    )
    
    # 3. 返回任务ID，等待回调
    return VideoTask(id=task.id, status="pending")

# 回调处理
@app.post("/callback/video")
async def video_callback(request: VideoCallbackRequest):
    """处理视频生成回调"""
    if request.status == "completed":
        # 下载视频到存储
        download_video(request.video_url, request.task_id)
        # 更新任务状态
        update_task_status(request.task_id, "completed")
        # 通知审核员
        notify_reviewer(request.task_id)
    else:
        # 处理失败
        update_task_status(request.task_id, "failed", request.error)
```

### 3.5 人工审核模块

class CostController:
    """成本控制器"""
    
    DAILY_BUDGET = 1000  # 日预算1000元
    
    def __init__(self):
        self.daily_cost = 0
        self.api_costs = {
            "deepseek-v4": 0,
            "glm-5.1": 0,
            "seedance-2.0": 0
        }
    
    def record_api_call(self, api_name: str, cost: float):
        """记录API调用成本"""
        self.api_costs[api_name] += cost
        self.daily_cost += cost
        
        # 检查是否超预算
        if self.daily_cost > self.DAILY_BUDGET:
            self.trigger_alert(f"日预算超支：{self.daily_cost}元")
    
    def get_cost_report(self) -> dict:
        """生成成本报告"""
        return {
            "daily_cost": self.daily_cost,
            "api_breakdown": self.api_costs,
            "remaining_budget": self.DAILY_BUDGET - self.daily_cost
        }
```
