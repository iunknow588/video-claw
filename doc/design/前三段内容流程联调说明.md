# 前三段内容流程联调说明

## 1. 范围

本说明只覆盖视频生产前的前三段内容流程：

```text
热点采集 -> 爆款DNA分析 -> 原创脚本生成
```

对应部门：

- `CSO`：热点采集
- `CCO`：爆款拆解与 DNA 分析
- `COO`：原创脚本生成

## 2. 当前可用状态

### 2.1 热点采集

当前接口：

```text
POST /api/hotspots/fetch
```

当前能返回完整热点条目，包括：

- `platform`
- `content_id`
- `title`
- `author`
- `author_id`
- `url`
- `cover_image`
- `video_url`
- `view_count`
- `like_count`
- `comment_count`
- `share_count`
- `category`
- `tags`
- `duration`
- `fetched_at`
- `source_mode`

说明：

- 当前 provider 仍是 `mock` 数据源
- `source_mode=mock` 用于明确标记该条热点并非真实平台抓取结果

### 2.2 爆款DNA分析

当前接口：

```text
POST /api/analysis
GET /api/analysis/hotspot/{hotspot_id}
```

当前可输出：

- `report_title`
- `framework_summary`
- `content_structure`
- `emotion_curve`
- `hook_design`
- `reusable_elements`
- `risk_warnings`
- `dna_report`

说明：

- 当前已经具备“结构化爆款DNA报告”的输出外形
- 模型真实返回可能出现字段形态差异，例如 `emotion_curve` 可能是字典，也可能是列表
- 当前响应层已经允许这种真实模型输出形态

### 2.3 原创脚本生成

当前接口：

```text
POST /api/scripts
GET /api/scripts
POST /api/scripts/review/{script_id}
```

当前可输出完整脚本内容，包括：

- `title`
- `content_type`
- `style`
- `topic`
- `duration`
- `hook`
- `cta`
- `tags`
- `scenes`
- `similarity_score`
- `api_cost`
- `script_bundle`

说明：

- 当前脚本内容是模型真实生成，不是占位文本
- 但当前脚本生成模型仍是 `XFYUN_MAAS` 路线，尚未切到 `GLM-5.1`

## 3. 联调命令

### 3.1 只测试前三段

```powershell
python src\scripts\front_flow_smoke.py
```

### 3.2 测试整条 API 流程

```powershell
python src\scripts\api_workflow_smoke.py
```

### 3.3 指定本地服务地址

两个脚本都支持：

```powershell
$env:LOBSTER_BASE_URL="http://127.0.0.1:8001"
python src\scripts\front_flow_smoke.py
```

## 4. 注意事项

- 本地冒烟脚本已显式禁用环境代理，避免 `localhost` 请求被代理转发后出现伪 `503`
- 当前前三段流程已经可以独立联调，不必依赖视频模块成功
- 视频模块仍应视为第四段单独验收
